A React.js web application for discovering and saving horror movies using The Movie Database (TMDB) API.



This is a 3-page React application that allows users to browse, search, and save their favorite horror movies. The application focuses exclusively on horror films and provides a dark, themed interface perfect for horror movie enthusiasts.

Home Page: Browse popular horror movies and search for specific titles
Movie Details: View detailed information about each movie including cast, ratings, and overview
Saved Collection: Save favorite movies to a personal collection with local storage persistence


Before running this project, make sure you have the following installed:

- **Node.js** (v14.0.0 or higher)
- **npm** (v6.0.0 or higher)



Open your terminal/command prompt and navigate to the project directory:
- cd path/to/i-love-horror-movies
- npm install
- npm run dev
- there are some dependancy errors that do not affect functionality during npm install (ignore for now)
